## 本地运行

```sh
git clone https://github.com/xiyou-linuxer/Plan.git xlg-plan
cd xlg-plan
# package-manager install pnpm
pnpm i
pnpm dev
```

需要的 VS Code 插件：

- ESLint：代码规范
- Prettier：代码格式化
- MDX：MDX 语法高亮

参考文档：

- [Nextra](https://nextra.site/)：文档主题配置
- [MDX](https://mdxjs.com/)：MDX 增强语法
