## 本地运行

```sh
git clone https://github.com/xiyou-linuxer/Plan.git xlg-plan
cd xlg-plan
# package-manager install pnpm
pnpm i
pnpm dev
# browser open http://localhost:3000
```

需要安装 `ESLint` 和 `MDX` VS Code 插件以规范代码并高亮语法。
